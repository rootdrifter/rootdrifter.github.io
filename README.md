# rootdrifter.github.io

Portfolio hub — the central landing page for the **rootdrifter** security portfolio,
served at <https://rootdrifter.github.io> via GitHub Pages (root user site).

One self-contained `index.html` — no build step, no frameworks; the only external
dependency is Google Fonts. It carries the shared design system used across all
project pages and links to each project's deep-dive page:

| Project | Page |
|---------|------|
| ironveil | <https://rootdrifter.github.io/ironveil/> |
| nullbyte | <https://rootdrifter.github.io/nullbyte/> |
| spectre | <https://rootdrifter.github.io/spectre/> |
| oracle | <https://rootdrifter.github.io/oracle/> |
| mirage | <https://rootdrifter.github.io/mirage/> |
| gauntlet | <https://rootdrifter.github.io/gauntlet/> |

Deployment: GitHub Pages, `main` branch, root directory.
